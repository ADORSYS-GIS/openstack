# OpenStack Installation Prerequisites

## Hardware Requirements

- CPU: 4+ cores
- RAM: 8GB minimum
- Storage: 100GB minimum
- Network: 1Gbps

## Software Requirements

- Ubuntu 22.04 LTS
- Python 3.8+
- Git
- Ansible 2.9+

## Network Setup

- Static IP for controller
- DNS resolution
- Open ports: 80, 443, 5000, 9696, 9292

## Storage Planning

- Root partition: 50GB
- OpenStack data: 50GB
- Swap: 8GB
